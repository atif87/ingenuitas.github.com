Google Summer of Code (GSoC) with SimpleCV

SimpleCV is happy to announce its participation in the 2012 Google Summer of Code. We are looking for up to three talented interns who want to become experts in computer vision and Python and work with a top-notch team on professional open-source development.

Who are we looking for?

We are looking for talented undergraduate and graduate students who want to help us change the landscape of industrial manufacturing by creating user-friend computer vision technologies. We are also looking for candidates that may not be from a computer vision background, but would like to apply computer vision to solve practical problems in the sciences and further their research domain. A good candidate student should have interest in any or all of the following computer science sub-disciplines: computer graphics, computer vision, artificial intelligence, machine learning, robotics, and signal processing. The student should also be scientifically minded - computer vision is a data driven field where results must be analyzed and quantified. Students should also be interested and motivated to construct the hardware necessary to collect imagery data, and build innovative applications that interact with the real world.

What skills will you need?

For our beginner and intermediate projects we expect each student to have some programming experience, and while we prefer that it would be in Python and C++, we do not require it. Python is really easy to learn and we will teach you everything you need to know, all we expect is that you have basic programming experience (i.e. at least one year of college programming courses). An understanding of computer graphics, computer vision, linear algebra, signal processing, and machine learning techniques will be helpful tools, but again they are by no means required.

Our advanced projects require that the student be familiar with C++ and Python and have some passing familiarity with linear algebra, signal processing, 3D geometry, and potentially computer graphics. For these projects we are looking for undergraduate seniors and graduate students.

What will your summer be like?

After an initial discussion with the student we will devise a project plan that meets the students learning objectives, and works to improve the functionality of SimpleCV. Since we aim to support all of the common operating systems students will be allowed to develop code on any platform and use whatever development environments suits them best. While not required, we strongly encourage students to spend the summer at our Ann Arbor, Michigan office at MakerWorks. MakerWorks fosters a large community of open-source hardware and software developers who can contribute to the students education. Our goal, for each of our students, is to be able to create an end-to-end example that will have real-world applications and show their code in action by the end of the summer. This can be accomplished by working on a single large project in the case of the advanced students, or as set of complimentary features for introductory students.

What sort of projects can I do?

Beginner Projects

  • Learn python by helping others learn python - we are always in need of good examples. The student will spend the summer writing examples of how to use SimpleCV with other python libraries. This will culminate in the student teaching a class using their examples at a hacker space or local university.
  • Basic extensions to SimpleCV - the student will propose and implement a set of features from our current open feature tickets plus any approaches the student might be interested in examining. For example, if the student is interested in color correction, they could implement a variety of color correction routines and then quantify the results. See our existing tickets to get a feel for current problems: https://sourceforge.net/p/simplecv/tickets/

Intermediate Projects

  • Intermediate extensions to SimpleCV - the student will propose and implement a set of complimentary features in SimpleCV working from our existing list of tickets and the students own interests. Example projects include improving our machine learning library, or integrating an augmented reality system with SimpleCV.
  • Panorama / Image Stitching - create a system that stitches successive images taken from an assembly line into a single image. The student will then extend this capability to other applications such as a panorama stitching system.
  • Improved Rendering - replace our current PyGame based rendering system with a system that can use either a Qt console or render interactively to a web-browser. Along these lines a student could choose to further integrate SimpleCV with the ipython web-notebook.
  • Tracking - the student will extend our current feature libraries to allow them to track objects temporally between frames successive frames. Complimentary methods include appearance matching, Kalman filtering, and OpenTLD (https://github.com/zk00006/OpenTLD).
  • Pick-to-Light System - for this project the student will create a pick-to-light system using a kinect sensor, a projector, and an optional camera. Pick-to-light systems drive order fulfillment systems at places like Amazon and McMaster-Carr. These systems cost in excess of $100 an item to install. We would like to write an open-source software package that provides this capability for free. Other applications include robotic user interfaces, interactive games, and artistic expression. See http://www.youtube.com/watch?v=emSHkvIgqSM

Advanced Projects

  • We would like to build and open-source hardware / software system for performing 3D laser scanning and structured light analysis. While there are source code repositories for such systems, none of them present a complete solution where a user can scan an object and generate a mesh model that can be printed on a 3D printed or used in a video game. We would like to create a set of python tools that make this process exceptionally easy. All project hardware will be consumer off the shelf and of minimal cost.
  • We are also willing to assist with any project that pertains to applying computer vision technology to fundamental research in the sciences. Interested graduate students are encouraged to work with their advisor to submit a research and development plan to create an open-source systems that expedites research or replaces costly proprietary software. For example, we have worked closely with graduate students at Harvard to create a system that tracks and quantifies research animals behavior.
